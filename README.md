# 📚 All Sortings

A collection of sorting algorithm implementations in **Python**. This repository is designed for beginners and students who want to understand how different sorting algorithms work.

## 🚀 Sorting Algorithms Included

- ✅ Bubble Sort
- ✅ Selection Sort
- ✅ Insertion Sort
- ✅ Merge Sort
- ✅ Quick Sort
- ✅ Counting Sort
- ✅ Cyclic Sort

## 📖 Algorithm Overview

| Algorithm | Best Case | Average Case | Worst Case | Space Complexity | Stable |
|-----------|-----------|--------------|------------|------------------|--------|
| Bubble Sort | O(n) | O(n²) | O(n²) | O(1) | ✅ Yes |
| Selection Sort | O(n²) | O(n²) | O(n²) | O(1) | ❌ No |
| Insertion Sort | O(n) | O(n²) | O(n²) | O(1) | ✅ Yes |
| Merge Sort | O(n log n) | O(n log n) | O(n log n) | O(n) | ✅ Yes |
| Quick Sort | O(n log n) | O(n log n) | O(n²) | O(log n) | ❌ No |
| Counting Sort | O(n + k) | O(n + k) | O(n + k) | O(k) | ✅ Yes |
| Cyclic Sort | O(n) | O(n) | O(n) | O(1) | ❌ No |

> **Note:** `k` represents the range of input values in Counting Sort.

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/all-sortings.git
```

2. Navigate to the project directory:

```bash
cd all-sortings
```

3. Run any sorting algorithm:

```bash
python bubble_sort.py
```

or

```bash
python quick_sort.py
```
